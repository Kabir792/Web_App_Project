# **Surveys**

## **Survey Objective**

To identify user expectations and required features.

## **Participants**

* Students: 30  
* Teachers: 5  
* Administrators: 3

## **Results**

### **Desired Features**

| Feature | Percentage |
| ----- | ----- |
| Grade Viewing | 90% |
| Attendance Tracking | 85% |
| Course Enrollment | 75% |
| Notifications | 70% |

## **Conclusion**

Most users prefer a centralized system with easy access to academic information.

# **User Journey**

## **Administrator Journey**

1. Login to the system.  
2. Access dashboard.  
3. Add new student.  
4. Update student information.  
5. Generate reports.  
6. Logout.

---

## **Teacher Journey**

1. Login to the system.  
2. View assigned students.  
3. Record attendance.  
4. Enter grades.  
5. Save updates.  
6. Logout.

---

## **Student Journey**

1. Login to the system.  
2. View profile.  
3. Check attendance.  
4. View grades.  
5. Logout.

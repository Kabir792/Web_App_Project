# **Technical Design Document (TDD)**

## **Overview**

The Student Management System is designed as a web-based solution for managing student records, attendance, courses, and grades.

## **Objectives**

* Centralized student information management.  
* Secure access control.  
* Efficient reporting.

## **Architecture Style**

Three-Tier Architecture

### **Client Layer**

* Browser-based interface

### **Server Layer**

* Application Server

### **Database Layer**

* MySQL Database

## **Major Components**

### **Authentication Module**

Responsibilities:

* Login  
* Logout  
* Session Management

### **Student Module**

Responsibilities:

* Create Student  
* Update Student  
* Delete Student  
* Search Student

### **Attendance Module**

Responsibilities:

* Record Attendance  
* View Attendance

### **Grade Module**

Responsibilities:

* Enter Grades  
* Update Grades

### **Reporting Module**

Responsibilities:

* Generate Reports

# **Functional Requirements**

## **FR-01 User Authentication**

The system shall allow users to log in using a valid username and password.

## **FR-02 Role-Based Access Control**

The system shall provide access according to user roles:

* Administrator  
* Teacher  
* Student

## **FR-03 Student Management**

The system shall allow administrators to:

* Add student records.  
* Update student information.  
* Delete student records.  
* Search student records.

## **FR-04 Course Management**

The system shall allow administrators to:

* Add courses.  
* Update course information.  
* Remove courses.

## **FR-05 Attendance Management**

The system shall allow teachers to:

* Record attendance.  
* Edit attendance records.  
* View attendance history.

## **FR-06 Grade Management**

The system shall allow teachers to:

* Enter grades.  
* Update grades.  
* View grade reports.

## **FR-07 Student Portal**

The system shall allow students to:

* View profile information.  
* View attendance records.  
* View grades.

## **FR-08 Reporting**

The system shall generate:

* Attendance reports.  
* Grade reports.  
* Student summary reports.

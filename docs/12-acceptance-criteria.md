# **Acceptance Criteria**

## **User Authentication**

### **Criteria**

* Users can login using valid credentials.  
* Invalid login attempts show an error message.  
* Users can logout successfully.

---

## **Student Management**

### **Criteria**

* Administrators can add students.  
* Administrators can update student information.  
* Administrators can delete student records.  
* Student data is saved successfully.

---

## **Attendance Management**

### **Criteria**

* Teachers can record attendance.  
* Attendance records are stored correctly.  
* Attendance history can be viewed.

---

## **Grade Management**

### **Criteria**

* Teachers can enter grades.  
* Grade information is stored correctly.  
* Students can view grades.

---

## **Reporting**

### **Criteria**

* Attendance reports can be generated.  
* Academic reports can be generated.  
* Reports display accurate information.

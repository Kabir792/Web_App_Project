# **User Stories**

## **Administrator**

### **Story 1**

As an administrator,  
I want to add student records,  
So that student information remains updated.

### **Story 2**

As an administrator,  
I want to update student information,  
So that records remain accurate.

### **Story 3**

As an administrator,  
I want to generate reports,  
So that I can monitor student performance.

---

## **Teacher**

### **Story 4**

As a teacher,  
I want to record attendance,  
So that student participation can be tracked.

### **Story 5**

As a teacher,  
I want to enter grades,  
So that student performance can be evaluated.

---

## **Student**

### **Story 6**

As a student,  
I want to view my grades,  
So that I can monitor my academic progress.

### **Story 7**

As a student,  
I want to check my attendance,  
So that I can stay informed about my participation.
